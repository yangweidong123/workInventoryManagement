# 库存管理系统 文档

## 项目概述

库存管理系统是一个前后端分离的 Web 应用，旨在实现商品库存的数字化管理，支持单品维护、批量导入、套餐组合及毛利率计算功能。

**目标用户**: 库存管理人员、采购人员、运营人员

**核心价值**:
- 实现商品库存的数字化管理，减少人工操作错误
- 支持批量导入，提高数据录入效率
- 套餐组合功能支持灵活的商品搭配销售
- 实时毛利率计算，帮助制定定价策略

## 技术选型

| 类别 | 选择 | 理由 |
|------|------|------|
| 前端框架 | Vue 3 | 渐进式框架，生态丰富，支持组合式 API |
| UI 组件库 | Element Plus | Vue 3 官方推荐组件库，功能完善 |
| HTTP 客户端 | Axios | 支持拦截器、Promise API |
| 后端框架 | Spring Boot 2.7 | 稳定版本，社区成熟 |
| ORM 框架 | MyBatis-Plus | 简化 CRUD 操作，支持分页 |
| 数据库 | MySQL 8.0 | 关系型数据库，稳定可靠 |
| 文件存储 | MinIO/Base64 | MinIO 生产环境使用，Base64 开发环境可选 |
| 构建工具 | Maven + npm | 主流构建工具 |

## 核心功能

### 库存维护模块
- **目的**: 管理商品库存信息，支持单品录入和批量导入
- **描述**: 
  - 手动录入商品信息，支持图片上传
  - Excel 批量导入，单次最多 1000 条
  - 自动计算货值和重量
  - 款号唯一性校验

### 套餐维护模块
- **目的**: 支持商品组合销售，计算套餐毛利率
- **描述**:
  - 多选库存商品生成套餐
  - 支持设置套餐总价
  - 实时计算毛利率
  - 数据校验（禁止重复商品、套餐价不低于成本价）
